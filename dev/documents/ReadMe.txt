DESCRIPTION

Run, jump and slide through dangerous obstacles! You're on
the run trying to collect as many coins as possible in a
world with no end in sight... how long will you last?

---------------------------------------------------------

CONTROLS

ARROWS      :: Menu Navigation
ENTER/SPACE :: Menu Select
SPACE       :: Pause
UP ARROW    :: Jump
DOWN ARROW  :: Slide
